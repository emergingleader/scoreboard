# EL Strategy Scoreboard — Q2 2026

Live scoreboard for the Emerging Leaders 2026–2030 strategy execution.

## Stack
- React 18 + Vite
- No external UI libraries
- localStorage for session persistence

## Local development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub (`emergingleader/scoreboard`)
2. Go to [vercel.com](https://vercel.com) → Import Project → select this repo
3. Vercel auto-detects Vite — just click **Deploy**
4. Done. Every push to `main` auto-deploys.
